A test may be associated with many controls (many to many relation). When creating a test, it has to be associated to atleast one Control.

Each test can have many Finding.

A test has five basic aspects:

  • Title
    This field has a maximum of 200 characters.
  • Reference
    This field has a maximum of 200 characters.
  • Description
    This is a regular text field where there is no limit to the number of characters.
  • Control
    A test has to be associated to atleast one Control.
  • Work papers
    Work papers are attachments or file uploads where you can keep files associated to a test.

Warning

Workpapers store the files that you upload on the server and therefore consumes disk space. The space consumed by a file is related to the size of the file. Be careful not to run out of server space by uploading extremely large files.
