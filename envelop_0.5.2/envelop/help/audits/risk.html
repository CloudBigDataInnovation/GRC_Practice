A risk may be associated with many objectives (many to many relation). When creating a risk, it has to be associated to atleast one Objective.

Each risk can have many Control s.

A risk has five basic aspects:

  • Title
    This field has a maximum of 200 characters.
  • Reference
    This field has a maximum of 200 characters.
  • Description
    This is a regular text field where there is no limit to the number of characters.
  • Objective
    A risk has to be associated to atleast one Objective.
  • Work papers
    Work papers are attachments or file uploads where you can keep files associated to a risk.

Warning

Workpapers store the files that you upload on the server and therefore consumes disk space. The space consumed by a file is related to the size of the file. Be careful not to run out of server space by uploading extremely large files.
