ObjectiveÂ¶

Objective may be associated with many processes (many to many relation). When creating an objective, it has to be associated to atleast one Process.

Each objective can have many Risk.

An objective has five basic aspects:

  • Title
    This field has a maximum of 200 characters.
  • Reference
    This field has a maximum of 200 characters.
  • Description
    This is a regular text field where there is no limit to the number of characters.
  • Process
    An objective has to be associated to atleast one Process.
  • Work papers
    Work papers are attachments or file uploads where you can keep files associated to an objective.

Warning

Workpapers store the files that you upload on the server and therefore consumes disk space. The space consumed by a file is related to the size of the file. Be careful not to run out of server space by uploading extremely large files.
